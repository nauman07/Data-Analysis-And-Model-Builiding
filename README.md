# 📊 Data Analysis and Model Building — Diabetes Dataset

An end-to-end machine learning project that performs **exploratory data analysis (EDA)** and builds **predictive models** on the Pima Indians Diabetes Dataset from Kaggle.

---

## 📖 About

This project demonstrates the full data science pipeline — from data loading and exploration to preprocessing, visualization, and model building — using the well-known diabetes dataset. It's ideal for beginners looking to understand how to turn raw data into actionable predictions.

---

## 📂 Repository Structure

```
Data-Analysis-And-Model-Building/
├── Analysis.ipynb      # Full analysis and model building notebook
└── README.md
```

---

## 📦 Dataset

The dataset used is the **Pima Indians Diabetes Dataset**, publicly available on Kaggle.

🔗 [Download Dataset from Kaggle](https://www.kaggle.com/mathchi/diabetes-data-set)

**Features include:** Pregnancies, Glucose, Blood Pressure, Skin Thickness, Insulin, BMI, Diabetes Pedigree Function, Age, and Outcome (target).

---

## ✨ What's Covered

- **Exploratory Data Analysis (EDA)** — Understanding data distributions, correlations, and patterns
- **Data Preprocessing** — Handling missing/zero values, normalization, feature engineering
- **Visualization** — Plots and charts for insights
- **Model Building** — Training classification models to predict diabetes
- **Model Evaluation** — Accuracy, confusion matrix, and performance metrics

## 🚀 Getting Started

### Install Dependencies

```bash
pip install pandas numpy matplotlib seaborn scikit-learn jupyter
```

### Run the Notebook

```bash
git clone https://github.com/nauman07/Data-Analysis-And-Model-Builiding.git
cd Data-Analysis-And-Model-Builiding
jupyter notebook Analysis.ipynb
```
